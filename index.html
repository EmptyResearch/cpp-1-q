Chapter 1 Quiz

Challenge:

    Write a program that will get two numbers from the user, and output the product (multiplication) of those two numbers.

Part 1:

    What code will initialize two variables (without setting them to anything) called "number1" and "number2" on the same line?


Correct! Nice Job!


Part 2:

    What code will output "Enter number one: ", then store user input in number1; and then output "Enter number two: ", and store user input in number2? (Code on four different lines)


Correct! Nice Job!


Part 3:

    What code will output the product of the two numbers (number1 and number2) in the form "The product of these two numbers is [product]"?


Correct! Nice Job!


Test it out!

    After these practice questions, try out your code at C++ Shell to see the result. Did you get what you expected? If so, you are ready to continue to Chapter 2!
